# notes-app

Command line tool to save and read notes

### Install dependencies
```
npm install
```

### Add a note
```
node app.js add --title="Test note" --body="This is a test note"
```

### List all notes
```
node app.js list
```

### Read a note
```
node app.js read --title="Test note"
```

### Delete a note
```
node app.js remove --title="Test note"
```
